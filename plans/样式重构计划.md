## 问题分析

### 原始现状
| 文件 | 行数 | 类数量 | 问题 |
|------|------|--------|------|
| `dialogs.scss` | 7000+ | 271 | 过于庞大，难以维护和查找 |
| 其他 scss 文件 | - | - | 开发新UI时约50%概率不查看，导致重复造轮子 |

### 根本原因
1. **发现性差**：开发者不知道有哪些可复用的样式类
2. **组织混乱**：dialogs.scss 成为"垃圾场"，什么都往里塞
3. **缺乏文档**：没有样式索引，只能靠搜索或记忆
4. **没有强制约束**：新增样式时没有检查清单

## 重构方案

### 阶段一：拆分 dialogs.scss（降低单文件复杂度）

将 dialogs.scss 按领域拆分为多个文件：

```
styles/overlays/
├── dialogs.scss           # 仅保留通用弹窗基础样式（.acu-modal-*）
├── dialogs-settings.scss  # 设置相关（.acu-settings-*）
├── dialogs-purge.scss     # 清除相关（.acu-purge-*）
├── dialogs-history.scss   # 历史记录（.acu-history-*）
├── dialogs-theme.scss     # 主题美化（.acu-theme-*, .acu-preset-*）
├── dialogs-update.scss    # 更新相关（.acu-update-*, .acu-manual-*）
├── dialogs-ball.scss      # 悬浮球外观（.acu-ball-*）
├── dialogs-widget.scss    # 看板组件（.acu-widget-*）
├── dialogs-tag.scss       # 标签相关（.acu-tag-*）
├── dialogs-font.scss      # 字体相关（.acu-font-*）
├── avatar-dialogs.scss    # 头像相关（已存在）
├── dashboard-dialogs.scss # 仪表盘相关（已存在）
└── tag-manager.scss       # 标签管理器（已存在）
```

**目标**：每个文件控制在 500 行以内

### 阶段二：提取通用 mixins（消除重复代码）

创建 `styles/_mixins.scss`，提取高频重复模式：

```scss
// styles/_mixins.scss

// ===== Flex 布局 =====
@mixin flex-center {
  display: flex;
  align-items: center;
  justify-content: center;
}

@mixin flex-between {
  display: flex;
  align-items: center;
  justify-content: space-between;
}

@mixin flex-col {
  display: flex;
  flex-direction: column;
}

// ===== 设置行 =====
@mixin settings-row {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 10px 0;
  border-bottom: 1px solid var(--acu-border);

  &:last-child {
    border-bottom: none;
  }
}

// ===== 按钮基础 =====
@mixin btn-base {
  display: flex;
  align-items: center;
  justify-content: center;
  padding: 8px 16px;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: all 0.2s ease;
  font-size: 13px;
}

@mixin btn-primary {
  @include btn-base;
  background: var(--acu-title-color);
  color: #fff;

  &:hover {
    filter: brightness(1.1);
  }
}

@mixin btn-secondary {
  @include btn-base;
  background: var(--acu-btn-bg);
  color: var(--acu-text-main);

  &:hover {
    background: var(--acu-btn-hover);
  }
}

// ===== 输入框 =====
@mixin input-base {
  padding: 8px 12px;
  border: 1px solid var(--acu-border);
  border-radius: 6px;
  background: var(--acu-input-bg);
  color: var(--acu-text-main);
  font-size: 13px;

  &:focus {
    outline: none;
    border-color: var(--acu-title-color);
  }
}

// ===== 颜色选择器 =====
@mixin color-swatch($size: 24px) {
  width: $size;
  height: $size;
  border-radius: 50%;
  border: 2px solid var(--acu-border);
  cursor: pointer;

  &:hover {
    transform: scale(1.1);
  }
}

// ===== 空状态 =====
@mixin empty-state {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 40px;
  color: var(--acu-text-sub);
  text-align: center;

  i {
    font-size: 32px;
    margin-bottom: 12px;
    opacity: 0.5;
  }
}
```

## 风险控制策略

1. **先加后删**：新文件先创建好，再删除旧文件中的重复
2. **注释标记**：在原位置标注 `// 已移至 xxx.scss`
3. **每步验证**：改一点就在酒馆里测试
4. **Git 兜底**：每完成一个组件就提交一次

### 预期效果

| 指标 | 重构前 | 重构后 |
|------|--------|--------|
| dialogs.scss 行数 | 7000+ | < 300（仅基础样式）|
| 单文件最大行数 | 7000+ | < 500 |
| 新 UI 重复造轮子率 | ~50% | < 10% |
| 样式可发现性 | 靠搜索/记忆 | 有索引文档 |

---

## 风险与注意事项

1. **不破坏现有样式**：拆分时保持类名不变，只改变文件组织
2. **渐进式迁移**：可以分批拆分，不需要一次性完成
3. **测试覆盖**：每拆分一个模块后，在酒馆中测试对应弹窗
4. **Mixin 不滥用**：只提取真正高频复用的模式，避免过度抽象
