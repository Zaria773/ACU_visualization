# 全局状态组件 (GlobalStatusWidget) 设计方案

## 1. 核心目标
为“全局/状态/环境”类表格设计专用的仪表盘组件，实现：
- **自动适配**：无需人工配置，自动识别时间、地点、天气、人物等字段。
- **沉浸式展示**：非编辑态隐藏表头，像原生 UI 一样展示。
- **智能布局**：根据字段类型和内容长度，自动选择最佳排版（流式/网格/独占）。

## 2. 智能解析逻辑 (Smart Parser)

组件将扫描表格的第一行数据（Global 表通常仅一行），按以下优先级解析（**注意：必须先解析时间，防止“场景时间”被误判为地点**）：

### 2.1 时间 (Time) - 副标题/状态栏
- **识别关键词**：`['时', 'Time', 'Date', 'Clock']`
- **类型细分**：
  - **流逝时间 (Passed)**：列名含 `['流逝', '经过', 'Passed', 'Elapse']` → 显示为 `+15m` 样式的徽章。
  - **上轮时间 (Last)**：列名含 `['上轮', 'Last', 'Prev']` → 灰色小字。
  - **当前时间 (Current)**：不含上述词 → 正常显示。
- **展示**：<i class="fa-clock"></i> 图标 + 时间文本。

### 2.2 地点 (Location) - H1 主标题
- **识别关键词**：`['场', '地', '位置', 'Loc', 'Place', 'Area', 'Zone']`
- **排除已识别的时间列**。
- **多列处理**：若存在多个地点列（如“国家”、“城市”、“区域”），按列索引排序，使用 ` › ` 连接。
- **展示**：<i class="fa-map-marker-alt"></i> 图标 + 地点文本（字号 `1.5rem`，加粗）。

### 2.3 天气 (Weather) - 视觉化图标
- **识别关键词**：`['天', '气', 'Weather', 'Climate']`
- **图标映射 (FontAwesome)**：
  - 晴/Sun/Clear → `fa-sun`
  - 阴/Cloud/Overcast → `fa-cloud`
  - 雨/Rain → `fa-cloud-showers-heavy`
  - 雪/Snow → `fa-snowflake`
  - 雷/Thunder/Storm → `fa-bolt`
  - 雾/Fog/Mist → `fa-smog`
  - 台风/风暴 → `fa-solid fa-hurricane`
  - 风 → `fa-solid fa-wind`
  - 默认 → `fa-cloud-sun`

### 2.4 人物 (People/Characters) - 标签化
- **识别关键词**：`['人', 'NPC', 'Chara', '在场', 'Role']`
- **自动分割**：检测内容中的分隔符 `[,，、/&]`，将长文本拆分为数组。
- **展示**：<i class="fa-users"></i> 图标 + Flex 布局的 Tag 列表。

### 2.5 其他属性 (Attributes) - 流式排版
- **识别**：所有未被上述规则捕获的列。
- **自动图标**：
  - 金钱/Money/Gold → `fa-coins`
  - 系统/System/Info → `fa-bell`
  - 状态/Status → `fa-heart-pulse`
- **排版策略**：
  - **Short (< 15 chars)**：`inline-flex` 胶囊样式，流式排列。
  - **Long (>= 15 chars)**：`flex width: 100%`，独占一行，适合系统通知或环境描写。

## 3. 组件结构与布局

```html
<div class="acu-global-widget">
  <!-- 1. 头部区域 (Header) -->
  <div class="acu-global-header">
    <div class="header-main">
      <h1 class="location-text">诺德林 › 贫民窟</h1>
    </div>
    <div class="header-side">
      <span class="time-current">12:00</span>
      <span class="time-passed badge">+15m</span>
    </div>
  </div>

  <!-- 2. 内容区域 (Body) -->
  <div class="acu-global-body">
    <!-- 天气模块 -->
    <div class="weather-block">
      <i class="fas fa-cloud-showers-heavy"></i> 暴雨
    </div>

    <!-- 属性流 (自动换行) -->
    <div class="attr-capsule"><i class="fa-coins"></i> 50G</div>
    <div class="attr-capsule"><i class="fa-heart"></i> 健康</div>

    <!-- 人物标签组 -->
    <div class="person-group">
      <span class="tag">爱丽丝</span>
      <span class="tag">鲍勃</span>
    </div>
  </div>

  <!-- 3. 底部区域 (Footer) -->
  <div class="acu-global-footer">
    <div class="system-notice">
      <i class="fa-bell"></i>
      系统：你感觉到周围的气氛变得凝重起来...
    </div>
  </div>
</div>
```

## 4. 交互设计

- **展示态 (Default)**
  - 隐藏标准 Dashboard 的标题栏（Title Bar）。
  - 组件背景透明或半透明，融入整体 UI。

- **编辑态 (Editing)**
  - 恢复显示标准标题栏（以便拖拽、删除）。
  - 内容区域略微缩小，添加虚线边框提示范围。

## 5. 配置与集成

- **模板注册**：在 `useDashboardStore` 中添加 `global` 模板。
- **关键词规则**：自动匹配含“全局”的表格。
- **默认参数**：`colSpan: 2` (全宽), `displayStyle: 'global'`。

## 6. 样式规划 (SCSS)

- 文件：`styles/components/global-status.scss`
- 变量使用：复用 `var(--acu-text-main)`, `var(--acu-title-color)` 等。
