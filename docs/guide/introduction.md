# 介绍

ACU 可视化表格是专用于黑科技数据库的可视化前端，让你可以通过直观的图形界面来查看和编辑聊天记录中存储的表格数据。

## 主要功能

### 📊 数据表格
以表格形式展示数据，支持：
- 搜索和过滤
- 排序
- 分页浏览
- 行内编辑

### 🎛️ 仪表盘
自定义看板，将重要数据聚合展示：
- 拖拽排序
- 多种布局
- 快速查看

### 🕸️ 关系图
可视化展示人物关系网络：
- 节点拖拽
- 自定义样式
- 关系连线

### 🏷️ 标签管理
全局标签库管理：
- 分类整理
- 批量操作
- 导入导出

### 🎴 抽签系统
增加游戏趣味的随机抽签：
- 多维度配置
- 运势权重
- 精美卡面

### 🎨 主题自定义
丰富的主题选项：
- 预设主题
- 自定义配色
- 深色/浅色模式
